Showcase: Death

Webcomic interactivo · Narrativa visual · Desarrollo web

Showcase: Death es un webcomic experimental desarrollado completamente para la web, que combina ilustración, texto, animación y sonido para crear una experiencia narrativa interactiva e inmersiva.

Características del proyecto

Narrativa interactiva

Estructura por capítulos con navegación personalizada y progresión narrativa visual.

Animaciones y efectos

Uso de animaciones suaves sincronizadas con el scroll para reforzar la atmósfera del relato.

Ambientación sonora

Sonido ambiental integrado que acompaña la lectura y mejora la inmersión.

Tecnologías utilizadas

HTML5
CSS3
JavaScript
GSAP
AOS
GitHub Pages

Decisiones de diseño y desarrollo

Narrativa web

El proyecto fue concebido como una experiencia nativa de la web, priorizando la lectura progresiva y la interacción sobre el formato tradicional de cómic.

Animación sutil

Las animaciones se implementaron de forma suave y no intrusiva para reforzar la atmósfera sin distraer del contenido narrativo.

Ambientación sonora

El sonido ambiental es opcional y controlado por el usuario, utilizado como recurso inmersivo y no como elemento invasivo.

Diseño oscuro

La paleta oscura y el uso de blur fueron elegidos para reforzar el tono introspectivo y mejorar la legibilidad del contenido visual.
